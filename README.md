Meow Daily - Android App

Aplikasi Android sederhana untuk manajemen harian pecinta kucing. Proyek ini dibuat sebagai tugas pemrograman mobile dengan fokus pada autentikasi pengguna dan persistensi data lokal.

Fitur Utama
- Registrasi User: Menyimpan data email, password, dan lokasi ke dalam `SharedPreferences`.
- Login System: Validasi akun yang sudah terdaftar dengan pesan error yang interaktif.
- Lupa Password: Fitur untuk memperbarui password lama berdasarkan email yang terdaftar.

Teknologi yang Digunakan
- Bahasa: Kotlin
- IDE: Android Studio
- Penyimpanan: SharedPreferences
- Layout: ConstraintLayout & ScrollView (Responsive)
